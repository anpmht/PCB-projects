# 8-channel-DAQ-PCB
this board can take 8 channel differential input voltage signals and read it accurately with the precision of 10uV  at sampling frequency of 5KHz.
### Amplifier- ADA4254
### ADC - ADSs1256
### Microcontroller - teensy_4.1
